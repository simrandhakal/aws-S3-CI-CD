Welcome to the S3 CI/CD Lab-v1

Overview

This lab demonstrates the integration of AWS S3 with a CI/CD pipeline.

Setup Instructions

  1. Clone the repository: git clone https://github.com/simrandhakal/aws-S3-CI-CD.git
  2. Navigate to the project directory: cd aws-S3-CI-CD
  3. Install dependencies: npm install
